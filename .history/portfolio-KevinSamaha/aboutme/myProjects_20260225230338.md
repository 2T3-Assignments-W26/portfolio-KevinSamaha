## Rainfall Statistics ##

**Description:** Had to design a program that lets the user enter the total rainfall for each month of a year into an
array. Validating the input, displaying an appropriate error message if/when the
user enters invalid input. Ihad to also make a function to calculate and display the total
rainfall for the year, another function to compute and return the 
average monthly rainfall, and another function to display the
months with the highest and lowest amounts of rain.

**Coded** in _C#_ in Vertual studio
```c#
// user inputs the month

    static Int32[] WeatherForEachMonth(Int32[] UserWeatherInput)
    {
        string[] Month = DateTimeFormatInfo.CurrentInfo.MonthNames;
        Console.WriteLine("what is the total rainfall for each month");
        for (int i = 0; i < 12; i++)
        {
            Console.WriteLine("Total rain fall in " + Month[i]);
            while (true)
            {
                //checks if input is a int if it is the function will return to main if not user will retry
                if (int.TryParse(Console.ReadLine(), out UserWeatherInput[i]))
                {
                    break;
                }
                Console.WriteLine("Try again only numbers allowed");
            }            
        }
        return UserWeatherInput;
    }
```
