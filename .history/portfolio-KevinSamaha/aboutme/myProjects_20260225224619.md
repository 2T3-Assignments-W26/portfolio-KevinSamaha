# Programing projects Year one #

## Rainfall Statistics ##

**Description:** Had to design a program that lets the user enter the total rainfall for each month of a year into an
array. You should fully validate the input, displaying an appropriate error message if/when the
user enters invalid input. You should use additional functions to calculate and display the total
rainfall for the year (do this entirely in the function), another function to compute and return the
average monthly rainfall (display the average in your main), and another function to display the
months with the highest and lowest amounts of rain.
